### Features

- There is a Turkey map in the page that you can use to mark. There is **Local Storage** support on this page. When you click to a few cities after, reload the page and as you can see your data has not been deleted ğŸ‰

- You can download the map as an image. You can use to &quot;Resmi indir&quot; button on Home page

**Collection Page**

1. Click to a few cities
2. Press the &quot;Linki kopyala&quot; button
3. Paste it the browser and enter the copied page as you can see, this page is a city collection page and you now have a link. You can share this link with your friends ğŸ˜

### âš™ï¸ How To Use

1.  Clone this repository

```bash
git clone https://github.com/yasinatesim/turkey-visited-map.git
```

2. Install the project dependencies


```bash

```
